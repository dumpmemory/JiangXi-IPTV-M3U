# JiangXi-IPTV-M3U
### ✳️简介

本项目初衷为给长辈家庭自用，只需在智能电视/电视盒子上安装DIPY直播APP，导入订阅链接后既可食用

免除送的垃圾ITPV盒子的卡顿、广告与~~有线电视费~~

只使用江西电信进行测试，主要内容为央视、卫视、港澳台和江西本地台

**本人尽量保证订阅链接长期不变，如果以上订阅链接暂时无法使用，请直接下载m3u文件**

#### 1️⃣基于安卓的智能电视/电视盒子

1. 在智能电视/电视盒子上安装DIPY直播APP

   - 不同设备安装第三方APP教程劳烦百度

2. 添加以下订阅链接实现自动更新
